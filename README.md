# Crowd-Funding-Service API
Crowd-Funding-Service REST API 프로젝트  

## 주요 기능
* User domain 구현
* Project domain 구현
* 페이징  

## 개발 환경  
* JDK 1.8
* SpringBoot 2.1.8.RELEASE
* JPA2
* Gradle 6.2.2
* DB : H2
* Springfox - swagger - ui 2.9.2
